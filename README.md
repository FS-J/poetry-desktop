# poetry-desktop

欢迎使用诗词桌面！

本程序由Go语言编写。

目前仅支持 Windows 。

## 简介
能够自动从网路上随机获取一首唐、宋诗或宋词，并将文本整合在背景图片上，最后自动设置为壁纸。

要查看帮助请在程序所在目录打开cmd命令行输入: poetry-desktop --help
